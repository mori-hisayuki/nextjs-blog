# NextJS Blog
勉強がてら現行のブログを移行できたらなくらいのモチベで作る

# 使用技術

| 種別         | 技術                 |
| ------------ | -------------------- |
| Code 管理    | Github               |
| CI/CD        | Github Action        |
| 開発環境     | Docker(devcontainer) |
| ホスティング | Cloudflare Pages     |
| ブログ機能   | NextJS               |

# Blog の作成コマンド
```
yarn create next-app --example blog-starter blog-starter-app
```
